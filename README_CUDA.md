# Решение СЛАУ итерационным методом Якоби на CUDA

Вычисления проводились на персональном компьютере с процессором Intel Core i5-12450H (12 потоков наблюдаемой параллельности), 12 МБ кэш-памяти L3 и дискретной видеокартой NVIDIA GeForce RTX 2050 с 
4 ГБ видеопамяти. Для GPU-вычислений использовался драйвер NVIDIA версии 535.274.02 с поддержкой CUDA 12.2. Операционная система — Linux (x86_64).

Условия остановки: относительная невязка $\leq$ $1e-4$.
